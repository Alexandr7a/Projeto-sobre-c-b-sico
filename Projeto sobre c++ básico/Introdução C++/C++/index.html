C++

C++ é uma das linguagens mais versáteis, permitindo a criação de tarefas básicas, como aplicações na linha de comando ou web,
até sistemas complexos de tempo real, que são amplamente utilizados no mercado financeiro.



Introdução

A linguagem de programação C++ foi lançada em 1985. Seu nome original era "C com classes", pois surgiu como uma expansão de C para incorporar o paradigma de programação orientado a objetos. Por outro lado, rapidamente demonstrado ser uma linguagem totalmente nova, ganhando uma marca e um nome.

C++ é uma linguagem amplamente utilizada que permite acesso a recursos de baixo nível, como ponteiros, bem como recursos de alto nível, como construção de novos tipos usando classes e recursos de programação genérica.


Elegância e desempenho são as únicas palavras com as quais podemos caracterizar o C++. Sua sintaxe é refinada e não é tão verbosa quanto às linguagens modernas de alto nível. O C++ suporta paradigmas de programação estruturada, orientada a objetos e genérica.

Além de fornecer acesso a recursos de baixo nível que melhoram o desempenho do código escrito, seus compiladores possuem recursos de otimização excelentes, tornando os programas escritos em C++ excelentes em termos de desempenho.


O C++ permite a criação de compiladores, interpretadores, bancos de dados, sistemas operacionais, aplicações para a Internet das Coisas, aplicações de aprendizagem automática e uma variedade de outras aplicações. Muitas bibliotecas gráficas podem ser usadas para construir interfaces de usuários gráficos, bem como bibliotecas especializadas para uma variedade de tarefas.


C++ é uma linguagem Viva. A linguagem é padronizada e atualizada com novos recursos, e é usada por empresas em todo o mundo. Dentre as empresas que utilizam C++, podemos citar Adobe, Microsoft, Apple, Mozilla, MySQL, EdeVerge, Bloomberg, Linkedin, NASA e NetFlix. Uma lista extensa, mas você pode encontrar uma grande quantidade de informações na internet.


Ao olhar para a história das linguagens de programação, vemos que várias linguagens surgiram, foram usadas e, em seguida, caíram em desuso. No entanto, o C++, apesar de quase quatro décadas desde seu lançamento, continua sendo usado. Devido às suas qualidades, elegância e desempenho, provavelmente continuará sendo usado por muitos anos.
