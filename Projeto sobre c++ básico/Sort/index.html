Sort C++

A função integrada sort() do C++ pode ser usada para classificar qualquer tipo de estrutura de dados em uma ordem específica. Ele é definido pelo arquivo de Cabeçalho <algorithm>.



Sort() na biblioteca de modelos padrão C++ :


Uma das operações mais elementares com dados é o sort(). Isso se refere à organização de dados de uma maneira específica, que pode ser crescente ou decrescente. O sort() é uma função interna C++ STL. O IntroSort é usado internamente por esta função. Uma combinação de QuickSort, HeapSort e InsertionSort é usada para repeti-lo mais profundamente. Ele usa normalmente o QuickSort. No entanto, se o QuickSort faz o particionamento injusto e leva mais tempo do que N * logN, ele muda para HeapSort e, quando o tamanho do array é muito pequeno, ele muda para InsertionSort.

Genericidade

sort é especificado genericamente, para que possa funcionar em qualquer contêiner de acesso aleatório e em qualquer forma de determinar que um elemento x desse contêiner deve ser colocado antes de outro elemento y .

Embora especificada genericamente, a classificação não é facilmente aplicada a todos os problemas de classificação. Um problema particular que tem sido objeto de alguns estudos é o seguinte:

  • Sejam A e B duas matrizes, onde existe alguma relação entre o elemento A[i] e o elemento B[i] para todos os índices válidos i .
  • Ordene A mantendo a relação com B , ou seja, aplique a B a mesma permutação que ordena A .
  • Faça o anterior sem copiar os elementos de A e B em uma nova matriz de pares , classificando e movendo os elementos de volta para as matrizes originais (o que exigiria espaço temporário O ( n ) ).

Uma solução para este problema foi sugerida por A. Williams em 2002, que implementou um tipo de iterador personalizado para pares de matrizes e analisou algumas das dificuldades em implementar corretamente tal tipo de iterador. A solução de Williams foi estudada e refinada por K. Åhlander.
